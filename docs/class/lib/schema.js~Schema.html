    public relation(name: string | object, fn: function) source

    Adds an instance relationship to documents constructed from Models compiled from this schema.

    Params:

    NameTypeAttributeDescription
    name string | object

    name for the specified relationsship, or an object of muliple relationsships.
    fn function

    relationsship to be added to the schema.

    Example:

    ingredientSchema.relation({
	'HAS_NUTRIENTS': Relations.hasMany('Nutrients'),
	'HAS_INGREDIENT': Relations.belongsTo('IngredientType'),
	'CONTAINS_INGREDIENT': Relations.belongsToMany('Recipies'),
})
